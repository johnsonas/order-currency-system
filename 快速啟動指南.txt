========================================
快速啟動指南
========================================

當前狀態：
✅ 資料庫：運行中
✅ 前端：運行中 (http://localhost:5173)
❌ 後端：未啟動

========================================
啟動後端（只需 3 步）
========================================

方法：使用 IntelliJ IDEA

1. 開啟 IntelliJ IDEA

2. 開啟專案
   - File → Open
   - 選擇：C:\Users\2500916\order-currency-system

3. 啟動應用程式
   - 找到檔案：src/main/java/com/example/ordersystem/OrderCurrencyApplication.java
   - 右鍵點擊 → Run 'OrderCurrencyApplication'
   - 或點擊類別旁邊的綠色三角形 ▶️

========================================
等待啟動完成
========================================

後端啟動成功會看到：
"Started OrderCurrencyApplication in X.XXX seconds"

然後就可以訪問：
- 前端：http://localhost:5173
- 後端 API：http://localhost:8080

========================================
如果 IntelliJ 啟動失敗
========================================

請檢查：
1. Java SDK 是否設定（File → Project Structure → Project SDK）
2. Maven 是否設定（File → Settings → Build → Build Tools → Maven）
3. 資料庫連線是否正常（應該沒問題，因為資料庫在運行）
